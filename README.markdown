This is a jQuery plugin to create a banner with rotating images

```javascript
var options = {
			"banner_urls" : [],
			"banner_space" : 5,
			"width" : 600,
			"height" : 200,
			"timeout" : 5000,
			"random" : false,
			"subtitles" : [],
			"text_color" : "#000000",
			"text_font" : "Arial"
		}
```
